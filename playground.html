Basic editing

Edit text

Double click to edit.

Formatting options appear in the top bar.



   

Copy - Undo

Work on all actions

Copy / Paste
ctrl C/V     ⌘ C/V 

Undo / Redo
 ⌘ Z  /  ⌘ ⇧ Z  
 ctrl Z / ctrl ⇧ Z

Images

Double click to pick the image among the uploaded ones.

Paste the link in the URL field to link external images.





Advanced editing

Insert section & components

 Click on a section or component to insert it after the active element.

Drag elements and drop according to the blue line that shows the insertion point.

Set properties

Select an element in the page.

Modify its available settings in the right panel. 

Resize columns

Drag border to set offset and width.  



Clone elements

 
 ⇧ ⌘ C

 ⇧ ctrl C

Copy elements

Copy and cut work on all elements and between pages.

Paste, inserts after the active element.

Tree actions

Select or sort elements

The tree is where to go to add/delete childs as pictures in carousels

Press Esc to select the element's parent

Clone elements

 
 ⇧ ⌘ C

 ⇧ ctrl C

Work on childs

In the tree you can select, add or remove childs as carousel's slides or navbar elements.

Sort elements

The tree has many features:

Show the selected element name

Select or sort elements

Add/delete nested childs (as pictures in carousels or elements in navbars)

Work on source

Select an element to get at its HTML source

Dragging elements creates the related snippets in the HTML source. Every action on the visual editor updates the source.

Add custom rules in the Saas editor

The Saas editor simplifies adding custom rules. Saas  is compiled before being saved.
